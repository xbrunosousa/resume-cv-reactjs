## Uso & Instalação (NPM):
Em sua linha de comando digite:
```sh
$ git clone https://github.com/xbrunosousa/resume-cv-reactjs.git && cd resume-cv-reactjs
$ npm i
$ npm start
```
Acesse a aplicação em `http://localhost:3000` 👨🏻‍💻
